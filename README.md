Painel Administrativo para controle de E-commerce desenvolvido com React e Tailwind. Interface intuitiva para gerenciamento de produtos (CRUD), controle de estoque em tempo real, visualização de pedidos e dashboard de vendas integrado à API.

Este é o painel de gerenciamento exclusivo para administradores da plataforma. Ele permite o controle total sobre o inventário, pedidos e configurações da loja, oferecendo uma interface limpa e eficiente para a gestão do negócio.



## 🛠️ Tecnologias
- **React.js** (Biblioteca principal)
- **TypeScript** (Tipagem e segurança de dados)
- **Tailwind CSS** (Design ágil e responsivo)
- **Axios** (Comunicação com o Backend)
- **Context API** (Gerenciamento de estado global)

## 🌟 Funcionalidades do Administrador
- **Gestão de Produtos (CRUD):** Adicionar, editar, listar e remover produtos com upload de imagens para o Cloudinary.
- **Controle de Pedidos:** Visualização detalhada de vendas e atualização de status de entrega.
- **Dashboard:** Métricas rápidas sobre o desempenho da loja e produtos mais vendidos.
- **Interface Segura:** Acesso protegido por autenticação JWT (integrado ao backend).

## 📁 Estrutura do Projeto
- **Components:** Elementos de UI reutilizáveis (Botões, Inputs, Tabelas).
- **Pages:** Telas de Login, Adicionar Itens, Listagem e Pedidos.
- **Assets:** Imagens e ícones do painel.

## ⚙️ Como rodar o projeto
1. Clone o repositório.
2. Instale as dependências: `npm install`.
3. Configure a URL base da API no seu arquivo de ambiente ou configuração.
4. Inicie o painel: `npm run dev`.
